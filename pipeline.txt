  ---                                                                                                     
  Step 0 — Run once: build the official test CSV
  python baeline_eval/build_official_test_set.py                                                              
  Filters the 377k-row MIMIC-CXR CSV down to the 5,159 images in the official test split (images the
  CXR-CLIP models never trained on). Saves to cxr_data/mimic_cxr_official_test.csv.                       
                                                                                                          
  ---
  Step 1 — Build the image gallery                                                                        
python baseline_eval/build_baseline.py --csv cxr_data/mimic_cxr_official_test.csv --output_dir ./eval_outputs/exp1 --max_samples 5000

 Creates symlinks for all 5,159 test images in baseline_output_official_test/paired_data/. No embedding  
  happens here.
                                                                                                          
  ---                                                           
  Step 2 — Run all model evaluations                                                         
    python baseline_eval/run_all_evals.py --paired_dir ./eval_outputs/exp1/paired_data --csv cxr_data/mimic_cxr_official_test.csv --output_dir ./experiments/my_exp165 --batch_size 64                                                
  
  For each model:
  - Loads images from paired_data/, encodes them with the model                                           
  - Encodes 105 text queries (14 single CheXpert labels + 91 two-label combinations)
  - Retrieves top-k images per query by cosine similarity                                                 
  - Ground truth: images where the queried label(s) = 1                                                   
  - Reports P@1, P@5, P@10 and R@1, R@5, R@10                                                             
                                                                                                          
  Saves per-model CSVs + summary CSVs + plots to ./experiments/official_test_eval/.                       
                                                                                                          
  ---                                                                                                     
  If you need to re-run only some models (e.g. GPU was busy):                                             
  CUDA_VISIBLE_DEVICES=2 python baseline_eval/run_all_evals.py --paired_dir ./baseline_output_official_test/paired_data --csv cxr_data/mimic_cxr_official_test.csv --output_dir ./experiments/official_test_eval

  Key details:                                                                                              
  - LoRA applies to 72 modules (attention projections + FFN), ~1.9% trainable params (2.9M / 154M)
  - Saves epoch_N_adapter.pt (small, ~11MB) each epoch + final_merged.pt at the end for eval                
  - Dataset builds captions matching the eval query format exactly ("atelectasis", "fracture and edema")
  - Missing image files are retried gracefully (common on network mounts)
